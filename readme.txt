=== KISS CMS Admin ===
Tags: admin, menu, cms
Requires at least: 2.1
Tested up to: 2.7-bleeding
Stable tag: trunk

Removes posts, links, design and comments from the admin menu. Intended for small installations, with non-technical users.

== Description ==

Makes WordPress administration as simple and focused as can be, for sites with just pages.

== Installation ==

1. Upload `kiss-cms-admin.php` to the `/wp-content/plugins/` directory.
2. Activate `KISS CMS Admin` through the 'Plugins' menu in Wordpress.
